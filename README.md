## > About

**Kraken** is a Wi-Fi auditing toolkit focused on **WPA/WPA2**.

It was built to:
- capture authentication handshakes
- test password strength through dictionary attacks

<br>

## > Install

```bash
git clone https://github.com/0xf0xy/Kraken.git
cd Kraken
sudo pip install .
kraken -h
```
